# D&D 5e Random Character Generator

Generates a lightweight character concept as a starting point for D&D players — not a replacement for a full character sheet, just a spark to build from.

Each character includes a race-appropriate name, class, alignment, ability scores (4d6 drop lowest), personality traits, a flaw, a bond, and a backstory hook.

Check it out at https://sidkumararaja.github.io/dnd-random-character-generator/
